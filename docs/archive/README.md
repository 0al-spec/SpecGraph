# Archived Source Material

This directory keeps historical source material that should remain available
for provenance, but should not appear as active proposal, review, or runtime
work.

- `proposal_sources/`: archived working drafts that have already been
  normalized into reviewable proposals.
- `research/`: exploratory notes that are not active SpecGraph proposal
  artifacts.
- `reviews/`: historical review notes whose actionable findings have already
  been routed into proposals, specs, or tooling work.
