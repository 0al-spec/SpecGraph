# Supervisor Stalled Run Salvage Protocol Source Draft

## Source Class

Working draft.

## Historical Concern

SpecGraph needed a governed recovery protocol for supervisor runs that stall
after mutating a child worktree, so useful bounded diffs are preserved as
reviewable salvage evidence rather than lost, silently accepted, or recovered
through ad hoc manual copying.
