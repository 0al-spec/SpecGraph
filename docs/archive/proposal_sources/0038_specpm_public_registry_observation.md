# SpecPM Public Registry Observation Source Draft

## Source Class

Working draft.

## Historical Concern

SpecGraph needed a read-only observation layer for the local SpecPM static
registry so operators could compare materialized bundle identities and
capabilities against registry-visible `/v0` payloads without assuming
publication, mutation, or stable public authority.

## Promotion Note

This archived source preserves the draft concern that was normalized into
`docs/proposals/0038_specpm_public_registry_observation.md`.
